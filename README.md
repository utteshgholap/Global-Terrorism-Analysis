# Global-Terrorism-Analysis
This Jupyter Notebook performs a comprehensive exploratory data analysis (EDA) and visualization of historical global terrorism data. Leveraging Python data science libraries, the notebook uncovers structural trends, regional distributions, and specific chronological patterns within non-state geopolitical risk data.
📊 Key Analytical Highlights

    Organizational Profiling: Uses horizontal bar charts to map out and rank the most active historical militant and insurgent groups globally (e.g., Taliban, ISIL, Shining Path), highlighting the long-tail distribution of political violence.

    Temporal & Cyclical Trends: Features dense matrix heatmaps tracking monthly incident frequencies across nearly five decades. This allows for clear root-cause identification of isolated tactical surges (e.g., the August 2001 or June 2007 anomalies) versus macro-level systemic shifts.

    Analytical Approach: Demonstrates end-to-end data processing—including data cleaning, aggregation using pandas/numpy, and advanced presentation-grade plotting to translate raw geopolitical figures into actionable risk-modeling insights.

🛠️ Tech Stack & Requirements

    Language: Python 3.x

    Core Libraries: pandas, numpy, matplotlib, seaborn (or plotly for interactive visualizations)
